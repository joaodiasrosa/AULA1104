code O que é HTML?

HTML é a linguagem usada para criar páginas na internet. Ela organiza textos, imagens, links e muito mais.

list Tags mais comuns

  • <h1> a <h6> - Títulos
  • <p> - Parágrafos
  • <a> - Links
  • <img> - Imagens
  • <ul> e <ol> - Listas

check_circle Vantagens do HTML

  1. Fácil de usar
  2. Funciona em todos os navegadores
  3. É gratuito
